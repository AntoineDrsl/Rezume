# rezume_web

## Before to start : 

Install all the packages:
```bash
cd rezume_web
```
```bash
npm init
```
Accept all
```bash
npm install
```

Create a database in MondoDB called "Rezume" with 4 tables: 
```bash
students
```
```bash
companies
```
```bash
cvs
```
```bash
posts
```
## To launch the application

Launch the backend (Node)
`cd server`
`node index.js`

Launch the frontend (AngularApp)
`cd rezume-web`
`ng serve --open`
