# rezume_web

## Before to start : 

Install all the packages:
```bash
cd rezume_web
```
```bash
npm init
```
Accept all
```bash
npm install
```

Create a database in MondoDB called "Rezume" with 4 tables: 
```bash
students
```
```bash
companies
```
```bash
cvs
```
```bash
posts
```
## To launch the application

Launch the backend (Node)
```bash
cd server
```
```bash
node index.js
```
Launch the frontend (AngularApp)
```bash
cd rezume-web
```
```bash
ng serve --open
```
